검색 엔진 최적화: Search Engine Optimaization (SEO)

google SEO guide 다운로드

  1. 명확하고 독창적인 타이틀 사용하기
    2. title 태그는 사용자와 검색 엔진에 특정 페이지의 주제를 알려준다.
    페이지의 콘텐츠를 정확하게 설명하는 제목을 붙여준다.
    페이지마다 동일한 태그가 아닌 독창적인 태그를 작성한다.

  2. description 메타 태그 활용하기
    3. 페이지에 description 메타 태그를 사용하면 검색 엔진에서 페이지 정보의 요약 내용을 볼 수 있다.
    1~2개의 문장이나 짧은 단락을 사용한다.

  3. 페이지의 URL 구조 개선하기
    4. 이해하기 쉬운, 알아보기 좋은 URL은 콘텐츠의 내용을 전달하는데 효과적이다.
    1~2개의 문장이나 짧은 단락을 사용한다.
    다른 URL을 가진 같은 내용을 담은 각각의 페이지가 있다면 301 리디렉션 또는 canonical 속성을 이용한다.
    <head>태그 안에 <link rel="canonical" href="1.html"> 를 입력하면 1.html로 이동한다.

  4. 바람직한 앵커 텍스트 사용하기
    5. 앵커 텍스트란 링크를 보낼 때 <a>요소의 내용에 해당하는 부분의 문자열을 뜻한다.
    윗 단락의 경우 "301 리디렉션"이 앵커 텍스트이다.
    URL, "문서", "클릭!" 과 같은 일반적인 앵커 텍스트는 사용x
    링크되는 페이지가 무슨 내용인지에 대한 기본적인 정보를 제공하자.
    CSS를 사용할 때 링크를 일반 텍스트와 헷갈리게 하지 않도록 하자.

  5. 이미지를 올릴 때 주의할 점
    6. 예시: <img src="images\logo.png" alt="윤세린 로고"> 파일의 이름과 경로 등의 의미를 분명하게 하는 것이 좋다.
    얼터너티브 텍스트(alt)를 제대로 작성하자.

  6. 적절한 제목 태그 활용하기
    7. 중요도, 계층관계에 따라 제목 태그(<h1> ~ <h6>)를 적절하게 사용하자.

  7. robot.txt를 활용하기
    8. naver.com/robot.txt google.com/robot.txt 을 주소창에 입력해보자.
    robot.txt는 웹사이트에 웹 크롤러같은 로봇들의 접근을 제어하기 위한 규약이다.
    민감한 콘텐츠를 숨기는 목적이나 보안 도구로써 robot.txt를 사용하지 않는것이 좋다.
    sitemap을 검색엔진에게 제공하는 것도 좋다.

  8. 페이지랭크란? 생활코딩HTML-페이지링크
    9. 페이지랭크가 높은 웹 페이지는 검색엔진에 의해 보다 많이, 잘 노출된다.
    다른 페이지에서 하이퍼링크를 통해 많이 유입되면 페이지랭크가 올라간다.
    페이지랭크가 높은 페이지에서 링크되는 페이지 역시 페이지랭크가 높아진다.
