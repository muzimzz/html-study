검색 엔진 최적화: Search Engine Optimaization (SEO)

google SEO guide 다운로드

  1. 명확하고 독창적인 타이틀 사용하기
    2. title 태그는 사용자와 검색 엔진에 특정 페이지의 주제를 알려준다.
    페이지의 콘텐츠를 정확하게 설명하는 제목을 붙여준다.
    페이지마다 동일한 태그가 아닌 독창적인 태그를 작성한다.

  2. description 메타 태그 활용하기
    3. 페이지에 description 메타 태그를 사용하면 검색 엔진에서 페이지 정보의 요약 내용을 볼 수 있다.
    1~2개의 문장이나 짧은 단락을 사용한다.

  3. 페이지의 URL 구조 개선하기
    4. 이해하기 쉬운, 알아보기 좋은 URL은 콘텐츠의 내용을 전달하는데 효과적이다.
    1~2개의 문장이나 짧은 단락을 사용한다.
    다른 URL을 가진 같은 내용을 담은 각각의 페이지가 있다면 301 리디렉션 또는 canonical 속성을 이용한다.
    <head>태그 안에 <link rel="canonical" href="1.html"> 를 입력하면 1.html로 이동한다.
